## 新增创意

### 046-词汇小火车 (Vocabulary Train) - 2D 语言类 ⭐⭐⭐⭐⭐
- **教育目标**：词汇学习，物体命名和认知
- **核心玩法**：
  - 物体命名（动物、水果、蔬菜、交通工具）
  - 图文配对（看图选词）
  - 听音辨物（听声音选图片）
  - 词汇扩展（同一个物品的不同名称）
- **特色**：
  - 四个主题关卡（动物、水果、蔬菜、交通）
  - 语音朗读单词
  - 可爱的物品图标
  - 完成后有词汇卡片收集

### 047-合作小帮手 (Cooperation Helper) - 2D 社交情感类 ⭐⭐⭐⭐⭐
- **教育目标**：培养合作意识和团队协作能力
- **核心玩法**：
  - 双人模式（两个玩家在同一设备上）
  - 合作搬运（一起搬运箱子到目标位置）
  - 合作拼图（轮流放拼图块）
  - 合作接物（一人接水果，一人递水果）
- **特色**：
  - 需要两个孩子一起玩
  - 培养轮流意识
  - 培养沟通协调
  - 完成任务后一起庆祝

### 048-3D形状乐园 (3D Shape Park) - 3D 认知类 ⭐⭐⭐⭐⭐
- **教育目标**：3D形状认知，立体几何基础
- **核心玩法**：
  - 认识立体形状：球体、立方体、圆柱体、圆锥体、金字塔
  - 3D形状识别（点击正确的形状）
  - 3D形状配对（找到相同的形状）
  - 3D空间旋转（拖拽旋转观察形状）
- **特色**：
  - 真实的3D形状渲染
  - 可以拖拽旋转360度观察
  - 形状数量认知（数一数有几个面）
  - 日常生活物品形状关联（球→球体，箱子→立方体）

### 049-比较小能手 (Compare Master) - 2D 认知类 ⭐⭐⭐⭐⭐
- **教育目标**：比较能力，数学认知基础
- **核心玩法**：
  - 大小比较（大象vs老鼠）
  - 多少比较（3个苹果vs5个苹果）
  - 长短比较（长蛇vs短蛇）
  - 高矮比较（高楼vs平房）
  - 轻重比较（羽毛vs石头）
- **特色**：
  - 五种比较维度
  - 直观的视觉对比
  - 语音引导"哪个更大？"
  - 渐进式难度（2选1→3选1→4选1）

### 050-友谊分享 (Friendship Share) - 2D 社交情感类 ⭐⭐⭐⭐

  - 帮助朋友（朋友摔倒时扶起来）
- **特色**：
  - 情境化教学
  - 培养同理心
  - 正向反馈（分享后大家都开心）
  - 社交规则启蒙
