## 新增创意

### 2D 游戏（优先开发）
1. **034-比较小能手 (Compare Master)** - 认知类，比较能力
   - 大小比较：大象 vs 老鼠
   - 多少比较：3个苹果 vs 5个苹果
   - 长短比较：长蛇 vs 短蛇
   - 高矮比较：高楼 vs 平房
   - 轻重比较：羽毛 vs 石头

2. **035-模式小侦探 (Pattern Detective)** - 逻辑类，模式规律识别
   - 颜色模式：红蓝红蓝__
   - 形状模式：圆形方形圆形方形__
   - 数字模式：123123__
   - 大小模式：大大小小大大小__
   - 动物模式：猫狗猫狗__

3. **037-因果实验室 (Cause & Effect Lab)** - 逻辑类，因果关系推理
   - 按开关 → 灯亮
   - 浇水 → 花开
   - 按门铃 → 有人开门
   - 下雨 → 地湿
   - 天黑 → 开灯

### 3D 游戏（加强开发）
4. **040-3D形状乐园 (3D Shape Park)** - 认知类，3D形状认知
   - 认识立体形状：球体、立方体、圆柱体、圆锥体、金字塔
   - 3D形状识别：点击正确的形状
   - 3D形状配对：找到相同的形状
   - 3D空间旋转：拖拽旋转观察形状
   - 形状应用：球形是球、立方体是盒子

5. **041-3D积木搭建 (3D Block Builder)** - 创意/建构类，空间建构能力
   - 选择不同形状的积木（立方体、长方体、圆柱）
   - 拖拽积木搭建房子、塔、城堡
   - 自由创意模式
   - 倒影模式（镜像对称）

### 语言类游戏（补充开发）
6. **038-词汇小火车 (Vocabulary Train)** - 语言类，词汇学习
   - 物体命名：动物、水果、蔬菜、交通工具
   - 图文配对：看图选词
   - 听音辨物：听声音选图片
   - 词汇扩展：同一个物品的不同名称

7. **039-故事小火车 (Story Train)** - 语言类，故事排序与创作
   - 故事排序：把打乱的图片按正确顺序排列
   - 故事补全：选择合适的图片完成故事
   - 简单故事：三段式图片故事
   - 创意模式：自己排列图片创作故事
