# Baby Games 👶🎮

> 专为儿童设计的无状态教育游戏合集

## 🌟 项目简介

这是一个基于蒙特梭利教育理念的儿童游戏项目，所有游戏都是**无状态设计**，无需后端服务器，可以直接在浏览器中运行。

### 第一个游戏：井字棋 ♟️

适合4岁+儿童的经典益智游戏，帮助培养：
- 逻辑思维能力
- 策略规划能力
- 亲子互动时光

## 🎮 游戏特点

- ✅ **无状态设计**：所有游戏逻辑在前端完成
- ✅ **完全免费**：使用 GitHub Pages 免费托管
- ✅ **响应式设计**：支持手机、平板、电脑
- ✅ **适龄友好**：大按钮、简单操作、即时反馈
- ✅ **亲子互动**：家长和孩子可以一起玩

## 🚀 快速开始

### 在线游玩

访问 GitHub Pages 链接（部署后会显示在这里）

### 本地运行

1. 克隆仓库：
```bash
git clone https://github.com/mayf3/baby-games.git
cd baby-games
```

2. 直接打开 `index.html` 文件，或使用本地服务器：
```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx serve

# 然后访问 http://localhost:8000
```

## 📋 游戏规则

### 井字棋
1. 两名玩家轮流在 3×3 的格子中放置符号（X 和 O）
2. 先将三个相同符号连成一条线（横、竖、斜）的玩家获胜
3. 如果格子填满仍无胜者，则为平局
4. 点击"重新开始"按钮开始新游戏

## 👨‍👩‍👧 亲子互动建议

- 家长可以先示范玩法，让孩子理解规则
- 鼓励孩子思考："这一步放哪里好呢？"
- 不要刻意让孩子赢，让孩子在失败中学习
- 庆祝孩子的每一次进步，无论是赢还是输

## 📚 开发说明

### 添加新游戏

1. 在项目根目录创建新的游戏文件夹
2. 包含必要的文件：`index.html`, `style.css`, `game.js`
3. 更新首页的游戏列表
4. 提交并推送，GitHub Pages 会自动部署

### 代码规范

- 使用语义化的 HTML
- CSS 使用 BEM 命名规范
- JavaScript 使用 ES6+ 语法
- 添加适当的注释
