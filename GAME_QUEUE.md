# 游戏开发队列

## 开发原则

- **2D 和 3D 交替开发**：周一三五做 2D，周二四六做 3D，周日精品优化或休息
- **3D 技术栈**：Three.js (WebGL)，纯浏览器运行，无需安装
- **仓库统一**：2D 和 3D 游戏都在 baby-games 仓库，共用游戏大厅
- **部署双渠道**：GitHub Pages（游戏大厅）+ itch.io（精选 3D 游戏单独发布）
- **自进化系统**：每完成一个游戏，自动分析覆盖度 → 识别缺失 → 生成新创意
